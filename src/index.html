Escrever números por extenso.

  • Negativos.
  • Decimais.

// Alguns exemplos.

var inteiro = extenso(123);

var decimal = extenso("1,23");

alert(inteiro);
alert(negativo);
alert(decimal);Executar

Instalação

Instale-o via linha de comando com:

  • NPM
  • Bower

$ npm install --save extenso

$ bower install --save extenso.js
