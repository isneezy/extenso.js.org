# Extenso.js.org

> Um site p/ o Extenso.js

Acesse-o em &lt;https://extenso.js.org/&gt;.

## Desenvolvimento

```
# Cloná-lo.
git clone https://github.com/theuves/extenso.js.org
cd extenso.js.org

# Instalar as dependências.
npm install
npm run bower

# Compilar LESS.
npm run less

# Aguardar alterações para compilar LESS.
npm run watch
```
