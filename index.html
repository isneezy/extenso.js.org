Escrever números por extenso.

Escreve números:

  • Negativos.
  • Decimais.
  • Com até 66 dígitos.
// Exemplos:

var x = extenso(123);
var y = extenso(-123);
var z = extenso("1,23");

alert([x, y, z].join("\n"));
Executar

Instalação

Via linha de comando faça:

npm install --save extenso
